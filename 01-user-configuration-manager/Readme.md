The user configuration manager is a python certification project on freecodecamp v9 python curriculum
it is a simple command line project which displays concepts like string manipulation, functions, dictionaries and conditional statements

FEATURES OF USER CONFIGURATION MANAGER ARE:
Add new settings
Update existing settings
Delete settings
View all saved settings
Automatic conversion of keys and values to lowercase
Clear success and error messages

AND THE FUNCTIONS ARE:
add_setting(settings, setting)
Adds a new key-value pair to the settings dictionary.
Returns an error message if the setting already exists.

update_setting(settings, setting)
Updates the value of an existing setting.
Returns an error message if the setting does not exist.

delete_setting(settings, key)
Removes a setting from the dictionary.
Returns an error message if the setting cannot be found.

view_settings(settings)
Displays all current settings in a readable format.
Returns "No settings available." when the dictionary is empty.
